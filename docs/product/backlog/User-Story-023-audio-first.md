# 4. Audio-First POI-Darstellung
**Als** Wanderer auf dem Fotopfad  
**möchte ich** beim Klick auf einen POI sofort die Möglichkeit haben, die Geschichte zu hören,  
**damit** ich das Smartphone wegstecken und die Umgebung genießen kann.

* **Akzeptanzkriterien:**
    * Das Bild wird als "Hero-Image" über die volle Breite ganz oben im Modal angezeigt.
    * Der Audio-Player rückt an eine prominente Position direkt unter das Bild.
    * Der Text wird auf einen kurzen "Teaser" (max. 3-5 Zeilen) reduziert.
    * Icons auf der Karte signalisieren durch ihr Design (z.B. Kopfhörer-Symbol), dass Audio-Inhalte verfügbar sind.
