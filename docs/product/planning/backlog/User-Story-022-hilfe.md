# 3. Hilfe-Konzept (Learning by Doing)
**Als** Nutzer auf der Route  
**möchte ich** die Funktionen der Karte (Standort, Layer etc.) direkt bei Bedarf erklärt bekommen,  
**damit** ich nicht vorab eine Anleitung auswendig lernen muss.

* **Akzeptanzkriterien:**
    * Ein dezentes Hilfe-Icon (`?`) wird direkt auf der Karte platziert.
    * Ein Klick darauf aktiviert ein Hilfe-Overlay mit Erklärungen zu den Icons.
    * Die Anleitung wird aus dem Start-Modal in dieses Overlay ausgelagert.
