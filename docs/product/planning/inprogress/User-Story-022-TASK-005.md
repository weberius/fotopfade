# User-Story-022-TASK-005: `locales/*/hilfeModalLi.md` – Markdown-Dateien für Hilfe-Inhalt anlegen

## Zugehörige Story
User-Story-022 – Hilfe: Bedienungshinweise über ein Karten-Control

## Beschreibung
Für jeden Namespace und jede unterstützte Sprache wird eine Markdown-Datei
`hilfeModalLi.md` angelegt. Diese Dateien enthalten die Bedienungshinweise,
die im modalen Dialog `#hilfeModalDiv` angezeigt werden.

Der deutschsprachige Standardinhalt ist für alle Routen identisch.
Moers erhält zusätzlich englische und französische Varianten.

## Inhalt – Deutsch (alle Routen)

```markdown
## Bedienungshinweise

- **Pfeil-Symbol (GPS):** Über das Pfeilsymbol können Sie jederzeit Ihre Position auf der Karte anzeigen lassen.
- **Route:** „Route" zeigt Entfernungen zwischen den Sehenswürdigkeiten.
- **Über das Projekt:** Unter „Über" finden Sie spannende Hintergrundinformationen zum Projekt.
- **Galerie:** In der „Galerie" können Sie ganz in Ruhe durch die schönsten Bilder stöbern.
```

## Inhalt – Englisch (moers/en)

```markdown
## How to use

- **Arrow symbol (GPS):** Use the arrow symbol to display your current position on the map at any time.
- **Route:** "Route" shows distances between points of interest.
- **About the project:** Under "About" you will find interesting background information about the project.
- **Gallery:** In the "Gallery" you can browse through the most beautiful images at your leisure.
```

## Inhalt – Französisch (moers/fr)

```markdown
## Guide d'utilisation

- **Symbole de flèche (GPS) :** Utilisez le symbole de flèche pour afficher votre position sur la carte à tout moment.
- **Route :** « Route » affiche les distances entre les points d'intérêt.
- **À propos du projet :** Sous « À propos » vous trouverez des informations de fond intéressantes sur le projet.
- **Galerie :** Dans la « Galerie » vous pouvez parcourir tranquillement les plus belles images.
```
